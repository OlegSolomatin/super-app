## Слои архитектуры

### Backend (Clean Architecture)
```
┌──────────────────────────────────────────────┐
│              API Layer (Routers)               │  ← FastAPI APIRouter
│  GET/POST/PUT/DELETE → валидация → сервис     │
├──────────────────────────────────────────────┤
│             Service Layer                      │  ← Бизнес-логика
│  Валидация, расчёты, агрегация, оркестрация    │
├──────────────────────────────────────────────┤
│            Repository Layer                    │  ← SQLAlchemy
│  CRUD, запросы, миграции                       │
├──────────────────────────────────────────────┤
│              Data Layer                        │  ← PostgreSQL + MinIO + Redis
│  Хранение, кэш, файлы                          │
└──────────────────────────────────────────────┘
```

### Flutter (Clean Architecture)
```
┌──────────────────────────────────────────────┐
│              UI Layer (Screens)                │  ← ConsumerWidget
│  Экраны, виджеты, навигация                    │
├──────────────────────────────────────────────┤
│            Provider Layer (Riverpod)           │  ← State management
│  AsyncValue: loading/data/error                │
├──────────────────────────────────────────────┤
│            Service Layer (Dio)                 │  ← API-клиенты
│  HTTP запросы, интерцепторы, токены            │
├──────────────────────────────────────────────┤
│              Model Layer                       │  ← fromJson/toJson
│  Dart-модели, DTO                              │
└──────────────────────────────────────────────┘
```
